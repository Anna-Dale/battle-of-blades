# Battle of Blades

A Beyblade X-inspired browser game — built with love for a 4-month celebration.

## Play Online

Once deployed to GitHub Pages, your game will be at:

**https://Anna-Dale.github.io/battle-of-the-blades/**

## Controls

- **Arrow keys** — Move your Beyblade
- **Space (hold + release)** — Charge and unleash a spin burst attack
- **Hearts** — Health (top of screen during battle)

Best of 3 rounds wins!

## Beyblades

All 17 Beyblade options are inspired by official [Beyblade X](https://beyblade.com/) tops including Sword Dran, Arrow Wizard, Scythe Incendio, Helm Knight, Soar Phoenix, and more.

## Local Development

```bash
cd ~/Projects/battle-of-blades
python3 -m http.server 8765
```

Open http://localhost:8765
